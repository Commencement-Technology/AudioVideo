# Audio_Video
Implemented Audio and Video Player in React Native

# For Android, Run : 
1- yarn install <br>
2- yarn android

# For IOS, Run : 
1- yarn install<br>
2- cd ios<br>
3- pod install<br>
4- cd ../<br>
5- yarn ios
